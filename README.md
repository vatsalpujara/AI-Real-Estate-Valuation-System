# 🏢 AI-Powered Real Estate Intelligence, Valuation & Investment Recommendation Platform

An end-to-end data science and machine learning architecture that eliminates speculative pricing errors and quantifies macroeconomic investment risk exposures across major high-growth tech corridors in India (Bangalore & Gurgaon footprints).

## 🚀 Key System Features (10 Deployed Pages)
* **Real-Time Property Valuation:** Utilizes an ensemble Random Forest Regressor to map continuous spatial features to accurate fair-market values with an 89.31% $R^2$ accuracy score.
* **Game-Theoretic Explainable AI (XAI):** Incorporates global SHAP summary tracking to make model predictions transparent and interpretable.
* **Programmatic Investment Recommendations:** Implements automated business rules to generate clear Buy / Hold / Sell metrics.
* **Structural Risk Engine:** Calculates local safety metrics and build degradation to categorize risk layers dynamically.
* **Multi-Year Price Forecasting:** Computes compounding tactical 1-year, mid-term 3-year, and strategic 5-year capital appreciation horizons.
* **🤖 Bonus Task AI Chat Assistant:** Features an interactive conversational chatbot that responds to strategic property queries using the project data rules.

## 🛠️ Technology Stack
* **Language/Environment:** Python 3.14, VS Code, Jupyter Notebooks
* **Core Data Engine:** Pandas, NumPy, Scikit-Learn, Joblib
* **Explainability & Graphics:** SHAP, Matplotlib, Seaborn
* **Deployment Web Hub:** Streamlit Framework
